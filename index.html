Archive Hunter SS

Telegram bot for archive fashion hunting: rare YSL, Dior Homme, Hedi Slimane, Japanese auctions, Mercari JP, and more.

View on GitHub Setup Guide

What this repo includes

  • Telegram bot and archive-hunting assistant
  • Memory training for archive designers and auction targets
  • Local DB fallback when Supabase is not configured
  • GitHub Pages website via `docs/`

Setup steps

  1. Copy the package files into your repo root.
  2. Create `.env` from `.env.example` and add your keys.
  3. Commit to the `main` branch.
  4. Push to GitHub and enable Pages for `main / docs`.
