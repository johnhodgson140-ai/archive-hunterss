What this repo includes

  • Telegram bot for archive fashion search and auction monitoring
  • Persistent memory/training notes for rare designers and auction targets
  • Local fallback storage so the bot works without Supabase
  • 24/7 hunt alerts and marketplace intelligence commands
  • GitHub Pages website powered by this repo

How it works

The bot uses Claude AI to interpret archive fashion search queries and generate actionable hunting advice. It stores user profiles, hunts, and watched listings locally, with optional Supabase support for cloud persistence.

Set up your own Archive Hunter

  1. Clone the repository to your PC.
  2. Run .\setup.bat in c:\archive-hunter.
  3. Open .env and fill in:
    TELEGRAM_BOT_TOKEN
    ANTHROPIC_API_KEY
  4. Leave SUPABASE_URL and SUPABASE_KEY empty if you do not want cloud persistence.
  5. Start the bot with .\.venv\Scripts\activate.bat and python bot.py.

Then message your bot on Telegram and use commands like /start, /help, /scan, and /hunt.

GitHub Pages website

This site is generated from the docs/ folder in the repository. After pushing to main, GitHub Actions will deploy the site automatically.

To enable the site, go to Settings > Pages in your GitHub repository and choose the main branch with the /docs folder.
